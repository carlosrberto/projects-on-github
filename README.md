# Simple React Starter Project

## What's included?

- React
- Hot Reload
- Sass
- PostCSS (`postcss-preset-env`)
- CSS Modules
- Jest
- Enzyme
- Husky (git hooks: lint on commit, tests on push)

## Setup

```
yarn
```

## Available commands

- `yarn start`
- `yarn build`
- `yarn lint`
- `yarn lint:fix`
- `yarn test`
- `yarn test:watch`
